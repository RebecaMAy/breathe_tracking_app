\doxysubsection{Detailed Description}
Colección de métodos estáticos para conversiones de bajo nivel. 

Proporciona herramientas para transformar tipos de datos primitivos (int, long) y objetos complejos (UUID, String) en arrays de bytes y viceversa. Útil para la comunicación con dispositivos BLE o protocolos binarios.
