\doxysubsection{Detailed Description}
Gestiona el envío de correos electrónicos en segundo plano. \texorpdfstring{$<$}{<}Void, Void, Void\texorpdfstring{$>$}{>}. 

Esta clase evita el bloqueo del hilo principal (Main Thread) al realizar operaciones de red. Utiliza la biblioteca {\bfseries{Java\+Mail}} para conectar con el servidor SMTP de Gmail.

{\bfseries{Características principales\+:}}
\begin{DoxyItemize}
\item Autenticación segura mediante "{}\+Contraseña de Aplicación"{} de Google.
\item Construcción de mensajes MIME multipartes (soporte HTML).
\item Inyección de estilos CSS para correos corporativos/profesionales.
\item Notificación visual (Toast) al finalizar el envío. 
\end{DoxyItemize}
