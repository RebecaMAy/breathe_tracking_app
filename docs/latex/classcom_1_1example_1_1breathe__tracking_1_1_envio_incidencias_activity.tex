\doxysubsection{Detailed Description}
Gestiona la interfaz y lógica para reportar incidencias de desconexión. 

Esta actividad se encarga de\+:
\begin{DoxyEnumerate}
\item Recibir datos del sensor fallido via Intent.
\item Pre-\/cargar un formulario con un mensaje y título sugerido.
\item Enviar un correo electrónico al administrador usando \doxylink{classcom_1_1example_1_1breathe__tracking_1_1_java_mail_a_p_i}{Java\+Mail\+API}.
\item Registrar la incidencia en la colección \textquotesingle{}incidencias\textquotesingle{} de Firebase Firestore.
\end{DoxyEnumerate}
