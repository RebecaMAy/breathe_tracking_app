\doxysubsection{Detailed Description}
Clase modelo para subir las lecturas de los sensores a Firebase Firestore. 

Esta clase define la estructura de datos que se guarda como documento dentro de la subcolección "{}mediciones"{} de cada sensor en Firestore, utilizada para mantener un historial de datos.

Esta clase actúa como un Data Transfer Object (DTO). Define la estructura exacta de los documentos que se almacenan en la subcolección {\ttfamily mediciones} de cada sensor.

\begin{DoxyNote}{Note}
El campo \textquotesingle{}fecha\textquotesingle{} está anotado con @\+Server\+Timestamp para asegurar que el servidor de Firebase registre la hora exacta de la subida, garantizando precisión en el historial. 
\end{DoxyNote}
