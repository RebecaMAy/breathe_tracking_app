\doxysubsection{Detailed Description}
"{}\+Tablón de Anuncios"{} reactivo de la aplicación (Singleton + Live\+Data). 

Esta clase implementa un almacén de datos centralizado que desacopla el productor de datos (\doxylink{classcom_1_1example_1_1breathe__tracking_1_1_sensor_tracking_service}{Sensor\+Tracking\+Service}) del consumidor (\doxylink{classcom_1_1example_1_1breathe__tracking_1_1_sesion_sensor_activity}{Sesion\+Sensor\+Activity}).

{\bfseries{Arquitectura\+:}} Actúa como un bus de eventos en tiempo real. Al usar {\ttfamily Mutable\+Live\+Data}, garantiza que\+:
\begin{DoxyEnumerate}
\item La UI siempre observe el último dato disponible.
\item No se produzcan fugas de memoria (Memory Leaks) si la actividad se destruye.
\item La actualización de la interfaz sea segura respecto a los hilos (Thread-\/safe) usando {\ttfamily .post\+Value()}.
\end{DoxyEnumerate}
