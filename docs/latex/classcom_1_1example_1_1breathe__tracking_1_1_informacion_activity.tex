\doxysubsection{Detailed Description}
Muestra el gráfico de datos históricos (CO2, O3, etc.) leídos de Firebase Firestore. 

Esta actividad gestiona la visualización de la evolución de contaminantes en un periodo de 24 horas. Su funcionamiento se divide en\+:
\begin{DoxyEnumerate}
\item Conexión a Firebase Firestore para recuperar la colección \textquotesingle{}datos\+\_\+grafico\textquotesingle{}.
\item Procesamiento de arrays de datos (un array por contaminante) en una lista de objetos Gas\+Data.
\item Renderizado de un gráfico de líneas interactivo usando la librería MPAndroid\+Chart.
\item Evaluación dinámica del riesgo (lógica de "{}semáforo"{} y carita) según los umbrales de cada gas.
\end{DoxyEnumerate}

Evalúa los datos mostrados y actualiza el icono y texto de estado. 

Lógica de clasificación\+:
\begin{DoxyItemize}
\item {\bfseries{Crítico (Face Sad)\+:}} 4 o más puntos superan el límite de peligro.
\item {\bfseries{Advertencia (Face Neutral)\+:}} Al menos 1 punto supera el límite de peligro (pico aislado).
\item {\bfseries{Alerta Moderada (Face Neutral)\+:}} 10 o más puntos superan el límite seguro.
\item {\bfseries{Aceptable (Face Happy)\+:}} Algunos puntos (1-\/9) superan el límite seguro.
\item {\bfseries{Óptimo (Face Happy)\+:}} Todos los valores están por debajo del límite seguro.
\end{DoxyItemize}
