\doxysubsection{Detailed Description}
Actividad principal que muestra la interfaz de usuario con los datos del sensor en tiempo real. 

Esta clase actúa como el "{}\+Dashboard"{} principal de la aplicación. Su arquitectura se basa en el patrón Observer a través de la clase Singleton \doxylink{classcom_1_1example_1_1breathe__tracking_1_1_tracking_data_holder}{Tracking\+Data\+Holder}.

{\bfseries{Funcionalidades principales\+:}}
\begin{DoxyEnumerate}
\item {\bfseries{Visualización de Datos\+:}} Muestra CO2, Ozono, Temperatura, Batería y Ubicación actualizándose automáticamente.
\item {\bfseries{Gestión de Servicios\+:}} Inicia y detiene el \doxylink{classcom_1_1example_1_1breathe__tracking_1_1_sensor_tracking_service}{Sensor\+Tracking\+Service} (Foreground Service).
\item {\bfseries{Feedback Visual\+:}} Cambia el color de las barras de progreso según umbrales de seguridad definidos.
\item {\bfseries{Gestión de Incidencias\+:}} Permite reportar problemas de conexión o hardware.
\end{DoxyEnumerate}
